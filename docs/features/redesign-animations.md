# Feature: Redesign & Animations

## Context

The current site is functional but not visually impressive. A portfolio targeting small business clients needs to demonstrate creativity and frontend skill on first impression — visitors decide in seconds whether to trust a developer with their project. Goal: more modern aesthetic, scroll-triggered animations, richer micro-interactions throughout.

## Scope

**In scope:**
- Visual overhaul of all pages (hero, about, projects, contact)
- Scroll-triggered entrance animations (likely Framer Motion)
- Richer micro-interactions: hover reveals, parallax, cursor effects
- Hero section redesign — the highest-impact single page
- Possible color palette refresh

**Out of scope:**
- Content changes (handled in case studies / services features)
- Structural/routing changes

## Open Questions

- Aesthetic direction: dark & moody developer, bold editorial, refined minimalist, or futuristic/glassmorphic? (Decide before starting)
- Add Framer Motion or stay CSS-only? (Framer Motion strongly recommended for scroll-triggered work)
- Keep the current teal accent (`#38B2AC`) or refresh the palette?
- Any reference portfolios or sites to draw inspiration from?
