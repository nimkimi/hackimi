# Feature: Privacy-Friendly Analytics

## Context

Without analytics, there's no way to know which pages convert, where visitors drop off, or whether the portfolio is working at all. Essential before any optimization effort. Privacy-friendly tools (Plausible, Fathom) avoid GDPR consent banners and don't require cookies — a good fit for a professional portfolio.

## Scope

**In scope:**
- Integrate Plausible Analytics (recommended: simple script tag, no cookies, GDPR-compliant by default)
- Track: page views, unique visitors, referral sources, top pages, contact form conversions
- Add a goal/event for contact form submission to measure conversion rate

**Out of scope:**
- Heatmaps, session recordings
- Self-hosting Plausible (use cloud unless there's a reason not to)

## Open Questions

- Plausible (~€9/mo) or Fathom (~€14/mo)? Both are solid — Plausible has a slightly better free trial.
- Make the dashboard public (sharable) or private only?
- Add custom events for other CTAs (book a call, CV download)?
